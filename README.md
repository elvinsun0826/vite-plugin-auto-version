# vite-plugin-auto-version

#### 介绍

在 vite 打包过程中自动生成项目版本，添加版本检测代码并通知或强制客户端刷新

本插件是 vite 专属插件，使用了 Vite 特有的插件钩子，使用时需注意

#### 软件架构

node + pnpm + typescript

#### 安装教程

pnpm add -D vite-plugin-auto-version
